# The Physics World

Before any physical reactions can occur in GameMaker you must first define the physics world in which they are going to happen. If you do not do this, GameMaker will default to the basic collision functions, and all other physics related code or settings will be ignored. You can set up the physics world from [The Room Editor](../../../../The_Asset_Editors/Rooms.md), but you can also set the same properties using code with the following functions:

-   [physics\_world\_create](physics_world_create.md)
-   [physics\_world\_gravity](physics_world_gravity.md)
-   [physics\_world\_update\_iterations](physics_world_update_iterations.md)
-   [physics\_world\_update\_speed](physics_world_update_speed.md)
-   [physics\_world\_draw\_debug](physics_world_draw_debug.md)
-   [physics\_draw\_debug](physics_draw_debug.md)
-   [physics\_pause\_enable](physics_pause_enable.md)
