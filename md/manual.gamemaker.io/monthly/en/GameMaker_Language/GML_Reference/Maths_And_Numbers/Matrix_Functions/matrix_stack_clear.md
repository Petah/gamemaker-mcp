# matrix\_stack\_clear

This function clears the matrix stack.

#### Syntax:

matrix\_stack\_clear();

#### Returns:

N/A

#### Example:

if (!matrix\_stack\_is\_empty())
{
    matrix\_stack\_clear();
}

The above code checks the matrix stack to see if it is empty and if it is not it clears it.
