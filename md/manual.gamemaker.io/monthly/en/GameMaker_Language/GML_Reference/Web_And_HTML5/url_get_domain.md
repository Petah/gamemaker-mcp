# url\_get\_domain

You can use this to get the domain where your HTML5 game is being played.

#### Syntax:

url\_get\_domain();

#### Returns:

[String](../../GML_Overview/Data_Types.md)

#### Example:

dom\_name=url\_get\_domain();

This will store the domain of your game in the variable "dom\_name".
