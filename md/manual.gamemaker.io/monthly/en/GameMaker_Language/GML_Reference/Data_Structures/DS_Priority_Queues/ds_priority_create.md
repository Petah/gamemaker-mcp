# ds\_priority\_create

This function creates a new priority queue data structure and returns a [Handle](../../../GML_Overview/Data_Types.md)Â to it. This value should be stored in a variable and used in all further function calls relating to the priority queue.

#### Syntax:

ds\_priority\_create();

#### Returns:

[DS Priority](ds_priority_create.md)

#### Example:

p\_queue = ds\_priority\_create();

This will create a new priority queue and assign its index id to the instance variable p\_queue.
