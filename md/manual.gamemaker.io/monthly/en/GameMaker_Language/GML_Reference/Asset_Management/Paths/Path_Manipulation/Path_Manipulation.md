# Path Manipulation

In GameMaker you are not just limited to the paths that you make using [The Path Editor](../../../../../The_Asset_Editors/Paths.md). There are a number of functions that can be used to manipulate and change those paths at run-time, or to even create new ones and then change and manipulate those.

The following functions are used for changing paths:

-   [path\_exists](path_exists.md)
-   [path\_add](path_add.md)
-   [path\_add\_point](path_add_point.md)
-   [path\_change\_point](path_change_point.md)
-   [path\_insert\_point](path_insert_point.md)
-   [path\_delete\_point](path_delete_point.md)
-   [path\_clear\_points](path_clear_points.md)
-   [path\_append](path_append.md)
-   [path\_assign](path_assign.md)
-   [path\_delete](path_delete.md)
-   [path\_duplicate](path_duplicate.md)
-   [path\_flip](path_flip.md)
-   [path\_mirror](path_mirror.md)
-   [path\_reverse](path_reverse.md)
-   [path\_rotate](path_rotate.md)
-   [path\_rescale](path_rescale.md)
-   [path\_set\_closed](path_set_closed.md)
-   [path\_set\_kind](path_set_kind.md)
-   [path\_set\_precision](path_set_precision.md)
-   [path\_shift](path_shift.md)
